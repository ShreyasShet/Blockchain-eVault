# LexVault — Blockchain eVault for Legal Records

Tamper-proof legal document storage using Ethereum + IPFS.

## Setup
1. Clone this repo
2. Run `npm install`
3. Copy `.env.example` to `.env` and add your Pinata JWT
4. Run `npm run dev`

## Stack
- React + Vite
- Ethers.js + MetaMask
- Pinata (IPFS storage)
- Solidity on Ethereum Sepolia testnet
